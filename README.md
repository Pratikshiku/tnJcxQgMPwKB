### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户（住户）、员工、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 房屋
5. 论坛
6. 通知
7. 个人中心
8. 房屋收藏


**员工功能：**
1. 登录
2. 个人中心
3. 修改密码
4. 个人信息
5. 报修管理
6. 房屋管理、房屋留言管理、房屋收藏管理
7. 小区环境管理
8. 缴费管理
9. 通知管理
10. 轮播图信息



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 报修管理
4. 基础数据管理（报修类型管理、房屋类型管理、环境类型管理、缴费类型管理、通知类型管理）
5. 论坛管理
6. 房屋管理、屋留言管理、房屋收藏管理
7. 小区环境管理
8. 缴费管理
9. 通知管理
10. 住户管理
11. 员工管理
12. 轮播图信息
